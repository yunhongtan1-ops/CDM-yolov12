# Dataset Access

## Public Dataset

The iSOOD dataset used in this study is publicly available and can be downloaded from:

https://zenodo.org/records/10903574

The iSOOD small-object subset can be generated using the script provided in this repository:

```bash
python make_isood_small_split.py
```

Please update the input and output paths in the script according to your local dataset directory.

## FSOOD Dataset

The FSOOD dataset was collected from real field inspection scenes and contains images acquired in specific local environments. Due to regional data management requirements, field-site sensitivity, and privacy-related restrictions, the full FSOOD dataset is not publicly released at this stage.

Researchers who are interested in accessing FSOOD for non-commercial academic research may contact the authors by email:

```text
zhaoji_ustl@outlook.com
```

To help us process requests, please include the following information:

- Name and affiliation
- Research purpose
- Intended use of the dataset
- Whether the data will be used only for non-commercial academic research
- Contact information

Dataset access may be provided after review and approval, subject to data protection requirements and applicable restrictions. The dataset should not be redistributed, used for commercial purposes, or used for identifying sensitive field locations without permission.

At this stage, a reduced anonymized subset is not provided because the images are closely associated with real inspection sites and local environmental contexts. We will consider releasing a more broadly shareable version in future work if the relevant data protection and anonymization requirements can be fully satisfied.
